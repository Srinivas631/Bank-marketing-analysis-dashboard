# Bank Marketing Campaign Analysis Dashboard

## 📌 Project Overview

This project analyzes bank marketing campaign data to identify key factors influencing customer conversion and campaign effectiveness.

## 📊 Tools Used

* Power BI
* Excel / CSV Dataset

## 📈 Key Insights

* Students have the highest conversion rate (~28%)
* Conversion peaks during specific months indicating seasonal trends
* Higher contact attempts do not always lead to better conversion

## 💡 Business Recommendations

* Focus campaigns on high-conversion customer segments
* Optimize number of contact attempts
* Target high-performing months for marketing
